# **App Name**: TradeWise

## Core Features:

- Daily Entry: Input daily trading profits and losses.
- Dashboard Display: Display key metrics such as total profit/loss, win rate, and average trade.
- Graphical Analysis: Visualize trading performance using interactive graphs, tracking progress over time.
- Profit/Loss Calculation: Automatically calculate and display daily, weekly, and monthly profit/loss totals.

## Style Guidelines:

- Primary color: Deep blue (#2962FF), reflecting stability and trust.
- Background color: Light gray (#F0F2F5) providing a neutral backdrop.
- Accent color: Bright green (#3DDC84) to highlight profits and positive trends.
- Font: 'Inter', a grotesque-style sans-serif providing a modern, neutral look for both headlines and body text.
- Use clear, professional icons for navigation and data representation.
- Clean and structured layout with a focus on readability and data visualization.
- Smooth transitions and subtle animations to enhance user experience and provide feedback on interactions.
